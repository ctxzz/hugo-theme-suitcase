## 📦 概要

AI Knowledge Suitcase 専用の Hugo テーマ「**Suitcase**」を実装しました。
このテーマは、AIと人間が共同で生成した知識を美しく可視化し、体系的に整理するためのナレッジベースです。

## ✨ 主な機能

### 🎨 デザイン
- **ミニマルで美しいUI** - 知識に集中できる静かな書斎のようなデザイン
- **ダークモード対応** - デフォルトでダークモード、ワンクリックでライトモードに切替可能
- **レスポンシブデザイン** - デスクトップ、タブレット、モバイルに完全対応
- **Inter フォント** - 美しく読みやすいタイポグラフィ

### 🔍 機能
- **高度な検索機能** - Fuse.js を使用した高速で正確な全文検索
- **Front Matter連携** - `topics`, `tags`, `confidence`, `stage`, `visibility` などのメタ情報を自動で可視化
- **関連記事の自動表示** - Hugo の関連記事機能を活用した知識のつながりの可視化
- **タグ・トピック一覧** - サイドバーでの整理された表示

### 🛠️ 技術的な特徴
- Hugo v0.120.0+ に対応
- Pure CSS（フレームワーク不使用）
- Vanilla JavaScript
- JSON 出力による検索インデックス
- タクソノミー対応（tags, topics）

### レイアウト
- `baseof.html` - ベーステンプレート

- `related.html` - 関連記事表示

- `main.css` - 基本スタイル（約700行）

- `favicon.svg` - ファビコン（スーツケースアイコン）

### ドキュメント
- `README.md` - 詳細な使用方法とインストール手順
